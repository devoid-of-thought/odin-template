# Created for project odin


## How to use this template

1. Open `package.json` and update the `name`, `description`, `repository.url`, `bugs.url`, `homepage`, and `author` fields.

2. Update this README with your actual project details!

3. Install dependencies:
```sh
npm install
```

4. Start the development server:
```sh
npm start
```

5. Build for production:
```sh
npm run build
```
